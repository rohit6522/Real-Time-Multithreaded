This project simulates different Operating System threading models
such as Many-to-One, One-to-One, and Many-to-Many.

## Features
- Visual simulation of threading models
- State transitions (New, Ready, Running, Blocked, Terminated)
- Semaphore-based synchronization
- Event logging

## Threading Models
- Many-to-One
- One-to-One
- Many-to-Many

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
